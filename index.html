This is a glTF viewer. It uses Cesium for rendering.
Files are read via HTML5 File API, so it's fast. No uploads to a server.
Drop a .gltf/.glb file or click to select one
Keyboard Commands:
(F) Toggle local reference frame
(B) Toggle bounding sphere
(H) Toggle this help and credits
Made with and Cesium by the folks at VirtualGIS. Fork this on GitHub.
